# Email Automation System

## Overview
This project automates sending personalized emails using Python and MySQL.

## Features
- CSV data import
- Bulk email sending
- Email personalization
- Status tracking (Sent / Failed / Pending)
- Logging system

## Tech Stack
- Python
- MySQL
- SMTP (Gmail)

## How to Run
1. Install dependencies
2. Update config.py with your credentials
3. Run:
   python main.py

## Project Structure
- main.py → main execution
- db.py → database operations
- email_sender.py → email logic
- logger.py → logging system

- UI dashboard
